# PROJETO GERADOR DE SENHAS

O projeto tem como objetivo gerar senhas aleatórias e seguras para serem usadas no dia a dia.
Inclui ainda um sistema configurável das senhas direto na página, segue as opções:

+ Incluir números
+ Incluir simbolos
+ Incluir letras maiusculas
+ Excluir caracteres semelhantes
+ Escolher o tamanho da senha entre 4 e 99




## INICIAR O PROJETO - DESENVOLVIMENTO

Use `npm i` para baixar as dependencias,
após isso, use `npm start` para inicia-los.
